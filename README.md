## 词条标准化(预览版)
### 总体规则
#### 词条格式
词条的标准描述格式为：
> \[触发:{触发方式定义}\]\[范围:{范围定义}\]{词条组别}#{词条关键字}({条件类别}:{条件定义}):数值1,数值2 **

举例:
> \[触发:回合开始\]回复#生命:-500
> 攻防效果#系数\(武功系别:御剑\):150
